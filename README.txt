Two APIs are used in this application: 
https://cryptingup.com/apidoc/#introduction - mostly because it returns assets in ranked order 
https://docs.coincap.io/ - for market details

Implemented functionality: 
Multi-page application with navigation. 
"Home" view is a list of top10 assets on the market 
"Details" view is available by clicking on any item in the list in "Home" view, or "Assets" view 
"Assets" is a list of all assets, with a search bar 
"Convert" view is allowing user to choose two currencies from the lists, choose the amount of currency to be converted, and perform the convertion
